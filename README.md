# 🧮 Multiplication Table Generator (CLI - Java)

A simple and efficient **Command Line Interface (CLI) application** built using **Java** that generates multiplication tables for any number entered by the user.

This project is designed to demonstrate **basic Java programming concepts**, including loops, user input handling, and structured CLI application development.

---

# 🚀 Features

✔️ Generate multiplication table for any number
📏 Custom table range selection
🖥️ Clean command-line interface
⚡ Fast and lightweight execution
🎓 Beginner-friendly Java project

---

# 🛠️ Technologies Used

* ☕ **Java**
* 💻 **Command Line Interface (CLI)**
* 🔁 **Loops (for loop)**
* 📥 **Scanner for user input**

---

# 📂 Project Structure

```
MultiplicationTableCLI/
│
└── MultiplicationTable.java
```

---

# ⚙️ How It Works

The program asks the user to:

1. Enter a number
2. Enter the range for the multiplication table

The program then generates and prints the multiplication table based on the provided input.

---

# ▶️ How to Run the Project

### 1️⃣ Clone the repository

```
git clone https://github.com/Dheerajparihar111/multiplicationTable
```

---

### 2️⃣ Navigate to the project folder

```
cd multiplication-table-java
```

---

### 3️⃣ Compile the Java program

```
javac MultiplicationTable.java
```

---

### 4️⃣ Run the program

```
java MultiplicationTable
```
